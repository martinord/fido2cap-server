Sobre WebAuthn y la autenticación sin contraseñas

WebAuthn es un nuevo estandard para la autenticación (verificar la identidad de un usuario) que pretende reemplazar las contraseñas con Llaves de Seguridad físicas.

Esta aplicación web permite usar una Llave de Seguridad para la autenticación sin contraseña. Eso permite que la autenticación sea un proceso más intuitivo y seguro.

Para registrarse, tu Llave de Seguridad debe ser dada de alta por un administrador.
